
1. Enter a Twitch VOD ID. 
	Ex: https://www.twitch.tv/videos/<VOD ID>


2. (Optional) Provide filters as a space-separated list of emote names. Use quotes for case-sensitive contains search.  
	Ex: "LUL" KEK = LUL, LULW, ... , KEK


3. (Optional) Adjust plot settings.

	Show max: The number of clickable points shown.

	Link offset: The intro time for timestamp links.

	Window: The period for stat collection. A larger window may be needed to less-active chats.


4. Download / Sync / Analyze

	-New VOD IDs will begin downloading.
 
	-Existing logs will "Sync" and continue downloading if incomplete.

	-Logs can be analyzed before / after download completes.


The resulting plot will show timestamp points which can be clicked to open / play.
